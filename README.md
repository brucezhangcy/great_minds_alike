A real-time group alignment game for office meetings. Don't aim for the *correct* answer ‚Äî aim for what the **majority** will say.

## How to Play

### For the Host (big screen)

1. Open the game at your URL
2. Type a question for the room ‚Äî e.g. *"What's the best programming language?"*
3. Click **Launch**
4. Share the join link that appears on screen with everyone in the room
5. Watch answer bubbles float in as people submit
6. When everyone has answered, click **Reveal the Great Minds ‚ú®**

### For Participants (mobile)

1. Open the join link the host shared
2. Enter your name
3. Think about what **most people** will answer ‚Äî not what you personally think is right
4. Type your answer and tap **Lock In My Answer**
5. Wait for the host to reveal

---

## Scoring

| Result | Points |
|---|---|
| Your answer matches the most common answer | **+4 points** |
| Any other answer | 0 points |

If multiple answers tie for the highest count, all of them award +4 points.

- Answers are **case-insensitive** ‚Äî *coffee*, *Coffee*, and *COFFEE* all count as the same
- The bigger the bubble, the more people said that answer
